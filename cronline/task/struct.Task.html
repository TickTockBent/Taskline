A schedulable asynchronous task.

Task represents a unit of work that can be executed on a schedule or manually. Tasks support retry logic, timeouts, statistics tracking, and pause/resume functionality.

§Examples

§Basic Task

use cronline::Task;

let task = Task::new(|| async {
    println!("Task executing!");
    Ok(())
}).with_name("My Task");

§Task with Configuration

use cronline::{Task, TaskConfig};
use std::time::Duration;

let task = Task::new(|| async {
    // Your task logic
    Ok(())
})
.with_name("Configured Task")
.with_config(TaskConfig {
    timeout: Some(Duration::from_secs(30)),
    max_retries: 3,
    retry_delay: Duration::from_secs(5),
    fail_scheduler_on_error: false,
});

§Task with State

use cronline::Task;
use std::sync::Arc;
use std::sync::atomic::{AtomicU32, Ordering};

let counter = Arc::new(AtomicU32::new(0));
let counter_clone = Arc::clone(&counter);

let task = Task::new(move || {
    let counter = Arc::clone(&counter_clone);
    async move {
        counter.fetch_add(1, Ordering::SeqCst);
        Ok(())
    }
});

where F: Fn() -> Fut + Send + Sync + 'static, Fut: Future<Output = Result<()>> + Send + 'static,

Creates a new task with the given async function.

The function will be called each time the task is executed. It should return a future that produces a Result<()>.

§Arguments
  • func - An async function or closure that returns Result<()>
§Examples
use cronline::Task;

// Simple task
let task = Task::new(|| async {
    println!("Hello from task!");
    Ok(())
});

// Task with error handling
let task = Task::new(|| async {
    // Do work...
    if false {
        return Err("Something went wrong".to_string().into());
    }
    Ok(())
});

pub fn with_schedule(self, cron_expr: &str) -> Result<Self>

Sets the cron schedule for the task (builder pattern).

§Arguments
  • cron_expr - A cron expression string (e.g., “0 * * * *”)
§Returns

Returns Ok(Task) if the cron expression is valid, or Err(CronlineError) if parsing fails.

§Examples
use cronline::Task;

let task = Task::new(|| async { Ok(()) })
    .with_schedule("*/5 * * * *") // Every 5 minutes
    .unwrap();
§Errors

Returns CronlineError::CronParseError if the cron expression is invalid.

pub async fn execute(&self) -> Result<()>

Executes the task with retry logic and timeout handling.

This method runs the task’s function, applies the configured timeout, handles retries on failure, and updates statistics.

§Returns

Returns Ok(()) if the task completed successfully (possibly after retries), or Err(CronlineError) if it failed after all retry attempts or timed out.

§Examples
let task = Task::new(|| async {
    println!("Running task...");
    Ok(())
});

match task.execute().await {
    Ok(()) => println!("Task succeeded"),
    Err(e) => eprintln!("Task failed: {}", e),
}

pub async fn cancel(&self)

Cancels a currently running task.

If the task is running, this will signal it to cancel gracefully. The task must be checking for cancellation to actually stop.

§Examples
let task = Task::new(|| async { Ok(()) });
// Start task execution in background
// ...
task.cancel().await;
Source

pub async fn pause(&self) -> Result<()>

Pauses the task so it won’t be executed by the scheduler.

A paused task remains in the scheduler but won’t be executed until it’s resumed with Task::resume.

§Returns

Returns Ok(()) if the task was successfully paused, or Err(CronlineError) if the task is currently running.

§Errors

Returns CronlineError::TaskExecutionError if the task is currently running.

§Examples
let task = Task::new(|| async { Ok(()) });
task.pause().await.unwrap();
Source

pub async fn resume(&self) -> Result<()>

Resumes a paused task.

A resumed task will be executed by the scheduler according to its schedule.

§Returns

Returns Ok(()) if the task was successfully resumed, or Err(CronlineError) if the task is not paused.

§Errors

Returns CronlineError::TaskExecutionError if the task is not paused.

§Examples
let task = Task::new(|| async { Ok(()) });
task.pause().await.unwrap();
// ... later ...
task.resume().await.unwrap();
